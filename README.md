## Developer
Lincoln Adura

## Description
An intermediate-level Python automation tool that automatically 
organizes files in any folder by sorting them into categorized 
subfolders based on their file type and extension. Built using 
Python's os and shutil modules.

## Features
- 📁 Automatically creates category folders
- 🖼️ Sorts Images (jpg, jpeg, png, gif, bmp, svg, webp)
- 🎥 Sorts Videos (mp4, mov, avi, mkv, flv, wmv)
- 📄 Sorts Documents (pdf, doc, docx, txt, xls, xlsx, csv)
- 🎵 Sorts Audio (mp3, wav, aac, flac, ogg)
- 🗜️ Sorts Archives (zip, rar, tar, gz, 7z)
- 💻 Sorts Code files (py, js, html, css, java, cpp)
- 📦 Moves everything else to Others
- 🔄 Handles duplicate filenames automatically
- ✅ Skips folders and hidden files
- 📊 Displays full organization summary

## How to Run
Make sure Python is installed, then run:
```bash
python file_organizer.py
```
